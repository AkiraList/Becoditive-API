Available Endpoints

Animals

"GET: /v1/animals/cat provides a random cat image and fact"
"GET: /v1/animals/dog provides a random dog image and fact"
"GET: /v1/animals/panda provides a random panda image and fact"

"GET: /v1/images/delete provides a delete meme with custom avatar"
"GET: /v1/images/ad provides a ad meme with custom avatar"

"GET: /v1/images/jail provides a jail meme with custom avatar"
"GET: /v1/images/gay provides a gay meme with custom avatar"
"GET: /v1/images/discordblue provides a discordblue meme with custom avatar"
"GET: /v1/images/wanted provides a wanted meme with custom avatar"
